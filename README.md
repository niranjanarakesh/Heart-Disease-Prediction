# Heart-Disease-Prediction

Tools & Technologies:         
Python • Pandas • NumPy • Matplotlib • Seaborn • Scikit-learn • Google Colab/jupyter Notebook


Task:

1.Performed end-to-end data preprocessing — handled null values, removed duplicates, and standardized column names

2.Conducted Exploratory Data Analysis (EDA) using correlation heatmaps, bar charts, pie charts, and line graphs to uncover patterns in patient health data

3.Applied Label Encoding and Standard Scaling for feature preparation

4.Built and trained a Decision Tree Classifier with an 80/20 stratified train-test split

5.Evaluated model performance using Accuracy Score


Key Insights

1.Male patients showed higher heart disease prevalence

2.Asymptomatic chest pain type was the strongest predictor of heart disease

3.Obesity and family history were confirmed as significant risk factors

4.Older male patients (i.e age 50+) had the highest concentration of cases
